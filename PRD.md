# Image Background Remover - MVP 需求文档

**版本：** v1.0  
**日期：** 2026-03-14  
**作者：** 焦媛

---

## 一、产品概述

一个在线图片去背景工具，用户上传图片后，系统自动去除背景，返回透明背景的 PNG 图片供下载。

**目标用户：** 电商卖家、设计师、普通用户  
**核心价值：** 免费、快速、无需安装软件

---

## 二、技术架构

```
前端 (React + Vite + Tailwind CSS)
    ↕ HTTP POST /remove-bg
后端 (Python FastAPI + rembg)
```

- 前端：React 18 + Vite + Tailwind CSS
- 后端：FastAPI + rembg（基于 U2Net 模型）
- 部署：前端 Vercel / 后端 Railway

---

## 三、MVP 功能范围

### 3.1 核心功能（必须有）

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持点击选择或拖拽上传，格式：JPG / PNG / WEBP |
| 自动去背景 | 上传后自动调用后端 AI 处理 |
| 结果预览 | 处理完成后展示原图 vs 去背景图对比 |
| 下载图片 | 一键下载透明背景 PNG |
| 处理状态 | 上传中 / 处理中 / 完成 / 失败 的状态提示 |

### 3.2 不在 MVP 范围内（后续迭代）

- 批量处理
- 用户账号系统
- 历史记录
- 手动精细调整边缘
- 付费/限额功能

## 五、API 接口

### POST /remove-bg

**请求：**
- Content-Type: `multipart/form-data`
- 参数：`file`（图片文件）

**响应：**
- 成功：返回 PNG 图片二进制流（`image/png`）
- 失败：返回 JSON `{"detail": "错误信息"}`

**限制：**
- 文件大小：≤ 10MB
- 支持格式：JPG、PNG、WEBP
